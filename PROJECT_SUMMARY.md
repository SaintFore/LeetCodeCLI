## 🎯 使用方法

### 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 初始化项目
python main.py init

# 3. 开始练习
python main.py practice
```

### 主要命令
- `python main.py init` - 初始化项目
- `python main.py practice` - 开始练习
- `python main.py stats` - 查看统计
- `python main.py list` - 列出题目
- `python main.py search <关键词>` - 搜索题目
